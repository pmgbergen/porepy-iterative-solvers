# Example scripts
This folder contains example runscript on how to set up the block preconditioners for fluid mass balance, momentum balance, hydro-mechanical, and thermo-hydromechanical systems. No effort has been made to ensure that the setups are physically meaningful (they are not), nor that the simulations can be run to completion (they possibly cannot).

Some guidance on how to modify the solver parameters can be found [here](https://github.com/keileg/solver_configuration_flow_setups/blob/main/flow_setup.py)

Better instructions on how to use the iterative solver framework will be provided in due course.
